# Java Console To-Do List

Простое консольное приложение для управления задачами на Java.

## Функции
- Добавление задач
- Просмотр всех задач
- Отметка выполненных
- Удаление
- Сохранение в файл

## Технологии
- Java 25
- Сериализация (ObjectInputStream/ObjectOutputStream)
- MVC структура: model / service / ui
